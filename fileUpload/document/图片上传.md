#### 图片上传到临时目录

##### url: `http://47.89.18.26:8292/web/fileUpload/picUpload`
请求方式 : `post`

参数名   | 含义    | 是否必须
-------|--------|-----
uploadFiles|图片文件|Y


###  返回值

参数名  | 含义
-------------|-------------

```json
{
  "code": "201",
  "msg": "success",
  "data": {
    "id": 1,
    "name": "2222222222222222222222222222",
    "url": "2222222222222222222222222222"
  }
}
```

#### 图片从临时目录到最终目录

##### url: `http://47.89.18.26:8292/fileUpload/tempToFormal`
请求方式 : `POST`

参数名    | 含义    | 是否必须
-------|--------|-----
module|模块名称|Y
mId|新增的实体id|Y
id|图片id（中间用，号隔开）|Y


###  返回值

参数名  | 含义
-------------|-------------


```json
{
  "code": "201",
  "msg": "success",
  "data": {
    "id": 1,
    "name": "2222222222222222222222222222"
  }
}
```
