# 市场活动获积分

## 当前文档版本: 1.1

--------------------------------
### 说明：市场活动获积分接口
#### 请求方式 : `post`
##### 请求接口:activity/{token}/{activityId}/{id}/{payMoney}


参数名    | 含义    | 是否必须
-------|--------|-----
userId       | 用户id     |Y
activityId   | 活动订单id |Y
id           |市场活动规则id|Y
payMoney    |支付金额  |Y


##### 返回值

```json
{
  "code": "success",
  "msg": "",
  "data": "sucess"
}
```
