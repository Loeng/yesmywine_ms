# 推荐人获积分

## 当前文档版本: 1.1

--------------------------------
#### 说明：推荐人获积分接口
#### 请求方式 : `post`
##### 请求接口:recommend/{token}/{reUserID}/{id}


参数名    | 含义    | 是否必须
-------|--------|-----
userId       | 用户id     |Y
reUserID     | 被推荐人的id    |Y
id           |推荐人规则id|Y


#####  返回值

```json
{
  "code": "success",
  "msg": "",
  "data": "sucess"
}
```
